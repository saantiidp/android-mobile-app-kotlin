# 📱 Proyecto Android — Desarrollo de Aplicaciones para Dispositivos Móviles

## 🧠 Descripción General

Este proyecto es una aplicación **Android** desarrollada en el marco de la asignatura **Desarrollo de Aplicaciones para Dispositivos Móviles** del **Grado en Ingeniería Informática de la Universidad Autónoma de Madrid (UAM)**, impartida en la Escuela Politécnica Superior.

La aplicación se ha construido siguiendo las **buenas prácticas del desarrollo Android moderno**, utilizando **Kotlin**, arquitectura basada en componentes de Android y un enfoque centrado en **usabilidad, mantenibilidad y calidad del software**.

El proyecto sirve como ejercicio completo de:
- Diseño de interfaces móviles
- Gestión del ciclo de vida de actividades y fragmentos
- Persistencia de datos
- Arquitectura reactiva con ViewModel y LiveData
- Integración con servicios externos (Firebase)
- Uso de bases de datos locales con ROOM

## 🎯 Objetivos del Proyecto

- Diseñar y desarrollar una **aplicación Android completa** usando Kotlin.
- Aplicar **arquitecturas modernas** de Android (ViewModel, LiveData, separación de capas).
- Implementar **interfaces de usuario atractivas y usables** con XML y Material Design.
- Gestionar correctamente el **ciclo de vida de actividades y fragmentos**.
- Implementar **persistencia de datos** usando ROOM y preferencias.
- Integrar **servicios externos** como Firebase.
- Asegurar la **calidad y mantenibilidad** del código.

## 🏗️ Arquitectura y Enfoque Técnico

El proyecto utiliza los componentes recomendados del ecosistema Android:

- **Lenguaje:** Kotlin
- **Arquitectura:**
  - ViewModel para gestión de estado
  - LiveData para observación reactiva de datos
  - Separación entre UI y lógica de negocio
- **Interfaz de Usuario:**
  - Layouts XML
  - Material Design
  - RecyclerView para listas eficientes
  - Data Binding para simplificar la gestión de vistas
- **Navegación:**
  - Intents entre actividades
  - Uso de Fragmentos para interfaces flexibles
- **Persistencia:**
  - ROOM (SQLite)
  - Preferencias para pares clave-valor
- **Servicios externos:**
  - Firebase
